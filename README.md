# Sales Dashboard | Power BI

## Overview

This project is an interactive sales analysis dashboard developed using Microsoft Power BI.

The purpose of this dashboard is to analyze sales performance,  customer behavior, and regional trends.

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Excel

## Key Metrics (KPIs)

- Total Sales
- Total Profit
- Total Orders
- Total Quantity

## Dashboard Features

- Sales performance analysis by state
- Top 10 performing states
- Sales analysis by category
- Yearly sales trend analysis
- Interactive filters and slicers

## Data Preparation

Data cleaning and transformation processes were performed using Power Query.

DAX measures were created to calculate business metrics and KPIs.
